### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/vaibhav1710/New-Portfolio.git
    cd New-Portfolio
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```


3. **Run the application:**

    ```bash
    npm start
    ```

    The application should be accessible at `http://localhost:3000`.
